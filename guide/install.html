Installation Instructions

Currently, Shift Swap is not optimized to be istalled and run as a web-app. To run Shift Swap, simply run the file using: java -jar ShiftSwap.jar

If this is unsuccessful, the program can simply be run by executing the jar file using the Java 8 Runtime.

If you are compiling from source, note you will need to set up a database as appropriate, and change the database connection information in Controller.java on line 37; the one provided is for demonstrative purposes.
